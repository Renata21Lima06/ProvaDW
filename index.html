PROVA

Faça uma pasta com seu nome e abra no vs code, adicione uma index.html, app.js e style.css.
- Na index.html declare os outros dois documentos;
- Na página html deve ter o seguinte layout:
* Uma título com h1 e um parágrafo com p
* Um botão.
* 3 figuras (pode ser repetida)
No CSS você fará a seguinte formatação: título tamanho 16px e cor azul. parágrado justificado. As figuras devem estar alinhas com flexbox utilizando between.
No botão deverá chamar uma função ao clicar para aparecer uma mensagem na tela "OlÁ JS". Essa função deve ser feita no app.js
